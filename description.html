About Project WrenchOS

WrenchOS is an addon that has the same functions as your average framework, but runs at 0ms server and client sided. The main goal of this project is to have sufficient addons for smaller frameworks like ox_core and ND_Core WrenchOS Comes with quite a few exports and alot of player data. with a really good database that stores everything scripters could need. WrenchOS has "jobs", player names (in progress), cash/banking exports, active playerids and fairly good systems that make it easier to script in smaller frameworks. This is all setup when players join your server for the first time, creating their own row in your mysql database that stores their data forever. This is basically a big utility script. !!!!!All exports can be used on client and server equally.!!!!! (This will be used in my future scripts)
