About Project WrenchOS

WrenchOS is a highly efficient addon that functions similarly to your average framework but runs at 0ms server and client-side, offering top-tier performance.

The main objective of this project is to provide sufficient addons for smaller frameworks like ox_core and ND_Core, making scripting easier and more streamlined.

WrenchOS comes packed with numerous exports and extensive player data, all supported by a robust database system that stores everything a scripter would need.

It includes features such as jobs, player names (in progress), cash/banking exports, active player IDs, and a set of well-optimized systems, making it a perfect fit for smaller frameworks.

All player data is stored automatically when they join the server for the first time, creating a dedicated row in your MySQL database that securely holds their information permanently.

In essence, WrenchOS is a powerful utility script that enhances the efficiency and capability of smaller frameworks.

Important: All exports can be used both on the client and server sides equally!

(WrenchOS will be integrated into my future scripts as well.)
