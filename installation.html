Installation

Dependencies:

You will need Ox_lib and OxMySql for this framework.

Follow the steps below to install WrenchOS:

  1. First, install the database using sql.sql. You will need to edit lines 19 and 20 to configure your database.
  2. Next, drop the WrenchOS folder into your server's resources.
  3. Afterward, configure WrenchOS to your liking by editing the config files as needed.
